# ARYA - Premium Barber House Website

Un sito web professionale, minimalista ed elegante per una barberia di alto livello.

## 🎨 Design

- **Palette**: Bianco, Nero e Grigio
- **Stile**: Minimalista, Luxury, Contemporaneo
- **Font**: Playfair Display (titoli), DM Sans (corpo), JetBrains Mono (labels)
- **Responsive**: Ottimizzato per mobile, tablet e desktop

## ✨ Features

### Sezioni Incluse:

1. **Home Page** - Hero accattivante con immagine B/N e tipografia moderna
2. **Chi Siamo** - Storia del proprietario/barbiere principale con statistiche animate
3. **Servizi** - Listino prezzi chiaro con 6 servizi diversi
4. **Portfolio** - Galleria fotografica con filtri interattivi (Tagli, Barbe, Styling)
5. **Team** - Presentazione dei 3 barbieri con foto monocromatiche
6. **Prodotti** - Vetrina prodotti premium per capelli e barba
7. **Prenotazione** - Sistema di booking online completo e intuitivo
8. **Contatti** - Mappa, orari, telefono, email e social
9. **Footer** - Navigazione e informazioni aggiuntive

### Funzionalità Tecniche:

- ✅ **100% Responsive** - Mobile-first design
- ✅ **Animazioni raffinate** - Scroll reveal, counter, hover effects
- ✅ **Custom cursor** - Effetto elegante su desktop
- ✅ **Preloader** - Animazione di caricamento iniziale
- ✅ **Sistema di prenotazione** - Form completo con validazione
- ✅ **Portfolio filtering** - Filtra per categoria
- ✅ **Smooth scrolling** - Navigazione fluida
- ✅ **Menu mobile** - Hamburger menu con animazione
- ✅ **Back to top button** - Pulsante per tornare in cima
- ✅ **Performance ottimizzate** - Lazy loading, debounce

## 🚀 Come Usare

### Opzione 1: Apertura Diretta
Semplicemente apri il file `index.html` nel tuo browser.

### Opzione 2: Server Locale (Consigliato)
```bash
# Con Python
python -m http.server 8000

# Con Node.js (npx)
npx serve .

# Con PHP
php -S localhost:8000
```

Poi visita `http://localhost:8000`

## 🖼️ Logo

Il file `logo.svg` è un placeholder. Per utilizzare il tuo logo PDF:

1. Converti `Logo_ARYA.pdf` in formato PNG o SVG
2. Salva il file come `logo.png` o `logo.svg` nella cartella del progetto
3. Il sito lo caricherà automaticamente

## 📝 Personalizzazione

### Modificare Testi
Tutti i testi sono nel file `index.html`. Cerca e modifica le sezioni corrispondenti.

### Modificare Colori
Nel file `styles.css`, aggiorna le variabili CSS in `:root`:
```css
:root {
    --black: #000000;
    --dark: #0a0a0a;
    --gray: #6b6b6b;
    --white: #ffffff;
}
```

### Modificare Immagini
Sostituisci gli URL delle immagini Unsplash con le tue foto reali.

### Aggiungere Social Links
Aggiorna i link `href="#"` nelle sezioni Team e Contact con i tuoi URL social.

## 🎯 SEO Base

Per migliorare il SEO, aggiungi nel `<head>` di `index.html`:

```html
<meta name="description" content="ARYA Premium Barber House - La migliore barberia per tagli di precisione e barba artistica">
<meta name="keywords" content="barberia, barbiere, tagli uomo, barba, styling, Milano">
<meta property="og:title" content="ARYA Premium Barber House">
<meta property="og:description" content="L'Arte del Stile Maschile">
```
